<li> 태그
<li></li> 태그는 list의 약자로, 목록을 만드는 태그입니다.
이 태그는 단독으로 쓰이지 않으며 <ul></li> 혹은 <ol></ol> 태그 내부에 들어갑니다.

<ol> 태그는 번호를 메겨 순서가 있는 (ordered list) 목록을 만듭니다.
<ul> 태그는 순서없이 모양으로 (unordered list) 목록을 만듭니다.
ex)
<ol>
	<li>목록1</li>
	<li>목록2</li>
</ol>

<ul>
	<li>목록1</li>
	<li>목록2</li>
	<li>목록3</li>
	<ol>
		<li>목록3-1</li>
		<li>목록3-2</li>
	</ol>
</ul>
